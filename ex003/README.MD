## 💽 Como baixar o projeto

1 - Clonar o repositório.

```bash
    $ git clone 
```

2 - Entrar no diretório.

```bash
    $ cd menu-bank 
```

3 - Instalar as dependências.

```bash
    $ yarn install 
```

4 - Iniciar o projeto.

```bash
    $ yarn start
```
